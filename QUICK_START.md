# Quick Start Guide

## Run Your First Test in 60 Seconds

### Step 1: Install Dependencies (30 seconds)
```bash
cd music-engines
pip install flask flask-cors librosa --break-system-packages
```

### Step 2: Start Server (10 seconds)
```bash
python3 server.py
```

**You should see:**
```
Initializing Music Engines Pipeline...
Pipeline initialized successfully.
 * Running on http://0.0.0.0:5000
```

### Step 3: Open Browser (10 seconds)
- Go to: `http://localhost:5000`
- You should see the TOTALITY ENGINE interface

### Step 4: Upload Audio (10 seconds)
- Drag any MP3 file onto the upload area
- Click "INITIALIZE ANALYSIS"
- Wait for results

**That's it. You just ran your first analysis.**
